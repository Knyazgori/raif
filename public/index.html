Пример работы формы

Для подключения формы на свой сайт, вставьте html код указанный ниже: 

<script defer="defer" src="<%= process.env.REACT_APP_API_URL %>/dist/static/js/main.js"></script>
<link href="<%= process.env.REACT_APP_API_URL %>/dist/static/css/main.css" rel="stylesheet">
